# 🚚 Sistema de Gestão de Entregas - Grande SP
Este é um sistema interativo baseado em mapas para gestão logística de entregas na Grande São Paulo. Ele permite mapear bairros por dia da semana, definir taxas de entrega específicas e consultar endereços de clientes em tempo real.

✨ Funcionalidades
Busca Automática de Bairros: Localiza as fronteiras geográficas oficiais de bairros e cidades da Grande SP via API Nominatim (OpenStreetMap).

Organização por Cores: Separação visual das áreas de entrega por dias da semana (Segunda a Sexta).

Gestão de Taxas: Permite atribuir valores de entrega individuais para cada região mapeada.

Consulta de Endereços: Barra de busca para localizar o endereço do cliente com um marcador, permitindo conferir instantaneamente em qual zona de entrega ele se encontra.

Persistência de Dados: Os dados são salvos automaticamente no navegador (LocalStorage).

Backup e Restauração: Função para exportar e importar arquivos JSON, garantindo que você nunca perca suas configurações.
